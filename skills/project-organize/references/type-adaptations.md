# Project Type Adaptations

Guidance for adapting the organize skill to different project types.

| Project Type | Likely Needs | Likely Skips |
|-------------|-------------|-------------|
| Hardware + firmware | GROUNDING, notebook, PROGRESS.md, project-context | — |
| Web app (greenfield) | GROUNDING, CLAUDE.md, .gitignore | notebook, PROGRESS.md |
| Web app (existing) | GROUNDING, stale doc fixes | skeleton directories |
| Homelab infrastructure | GROUNDING, PROGRESS.md | notebook, project-context |
| Patent / legal | GROUNDING (careful about content boundaries) | PROGRESS.md |
| Data pipeline | GROUNDING, project-context | notebook |
